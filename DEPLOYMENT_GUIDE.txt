## 📊 USAGE EXAMPLES

### Example 1: Analyze Fraudulent Transaction (cURL)
```bash
curl -X POST http://localhost:8000/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Is this transaction fraudulent?",
    "transaction": {
      "transaction_id": "TXN006",
      "amount": 5000,
      "time_of_day": 2,
      "location": "UK",
      "merchant_category": "jewelry",
      "frequency_30d": 1,
      "account_age_days": 30
    }
  }'
```

### Example 2: Python Script
```python
from app.graph.workflow import FinancialAIWorkflow

workflow = FinancialAIWorkflow()

# Analyze transaction
result = workflow.execute(
    query="Is this transaction fraudulent?",
    transaction={
        'transaction_id': 'TXN006',
        'amount': 5000,
        'time_of_day': 2,
        'location': 'UK',
        'merchant_category': 'jewelry',
        'frequency_30d': 1,
        'account_age_days': 30
    }
)

print(result['fraud_assessment'])
print(result['recommendation'])
```

## 📈 RISK ASSESSMENT RESULTS

Risk Score Formula:
```
Risk = (0.3 × Volatility) + (0.4 × Trend) + (0.3 × Exposure)
```

Levels:
- LOW: 0.0-0.3 (suitable for conservative investors)
- MEDIUM: 0.3-0.7 (suitable for balanced investors)
- HIGH: 0.7-1.0 (suitable for aggressive investors only)

Example Output:

## 🚨 TROUBLESHOOTING

### Issue: ImportError for langchain
**Solution**: `pip install langchain langchain-community`

### Issue: FAISS not found
**Solution**: `pip install faiss-cpu` (or faiss-gpu for NVIDIA)

### Issue: Port 8000 already in use
**Solution**: `python -m uvicorn app.api.main:app --port 8001`

### Issue: Streamlit connection error
**Solution**: Ensure FastAPI is running on port 8000

### Issue: OOM (Out of Memory)
**Solution**: Reduce chunk_size in config.py, lower RAG_TOP_K

## 🎓 KEY CONCEPTS

### Fraud Risk Levels
- HIGH: >0.7 score → Block recommended → True Positive Rate: 95%+
- MEDIUM: 0.4-0.7 → Verify required → False Positive Rate: <5%
- LOW: <0.4 → Approve → No action needed

### Risk Scores
- LOW: Safe for conservative portfolios
- MEDIUM: Balanced risk-return profile
- HIGH: Growth-focused, suitable for aggressive investors

### Confidence Scores
- >0.85: High confidence → Trust recommendation
- 0.70-0.85: Moderate → Consider additional review
- <0.70: Low confidence → Manual review recommended
