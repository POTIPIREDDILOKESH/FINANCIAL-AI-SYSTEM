## PART 3: RUNNING THE SYSTEM

### Option A: Quick Start with Streamlit (Easiest)

Run this single command:
```bash
streamlit run frontend/streamlit_app.py
```

Expected output:
```
You can now view your Streamlit app in your browser.

Local URL: http://localhost:8501
Network URL: http://XXX.XXX.XXX.XXX:8501
```

Open your browser and navigate to: http://localhost:8501

The app will load with:
- Dashboard interface
- Transaction scanner
- Portfolio analyzer
- Market research tools
- History viewer

### Option B: Full Stack with FastAPI + Streamlit (Production)

You need two terminal windows:

#### Terminal 1 - Start FastAPI Backend Server
```bash
python -m uvicorn app.api.main:app --reload --host 0.0.0.0 --port 8000
```

Expected output:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete
```

The backend will be available at:
- API: http://localhost:8000
- Docs: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

#### Terminal 2 - Start Streamlit Frontend
```bash
streamlit run frontend/streamlit_app.py
```

Now you have:
- FastAPI backend on port 8000
- Streamlit frontend on port 8501
- Both working together

```json
  "risk_score": 0.456,
  "volatility_risk": 0.25,
  "trend_risk": 0.50,

```

Risk Levels:
- LOW (0.0-0.3): Safe for conservative investors
- MEDIUM (0.3-0.7): Balanced approach
- HIGH (0.7-1.0): Aggressive investors only

### Recommendation Result
```json
{
  "decision": "REJECT",
  "reasoning": ["Multiple fraud indicators detected"],
  "risk_level": "HIGH",
  "confidence_score": 0.95
}
```

Decisions:
- BUY/SELL/HOLD: For investments
- APPROVE/REJECT: For loans/credit
- VERIFY: For verification-needed transactions

## PART 7: TROUBLESHOOTING

### Problem: "Python 3.10+ required"
**Solution**: Install Python 3.10 or higher
```bash
python --version  # Check current version
# Download from https://www.python.org/downloads/
```

### Problem: "Port 8000 already in use"
**Solution**: Use a different port
```bash
python -m uvicorn app.api.main:app --port 8001
```

### Problem: "ImportError: No module named 'app'"
**Solution**: Make sure you're in the project root directory
```bash
cd financial-ai-system
python test_system.py  # Run from project root
```

### Problem: "ModuleNotFoundError: No module named 'streamlit'"
**Solution**: Install dependencies again
```bash
pip install -r requirements.txt
```

### Problem: Streamlit can't connect to API
**Solution**: Ensure FastAPI is running on port 8000
```bash
# Terminal 1: Start FastAPI
python -m uvicorn app.api.main:app --reload

# Terminal 2: Start Streamlit
streamlit run frontend/streamlit_app.py
```

### Problem: Low memory/slow performance
**Solution**: Modify config.py
```python
# Reduce these values:
RAG_TOP_K = 3  # Instead of 5
CHUNK_SIZE = 250  # Instead of 500
```
